Explicando Links

Existem links externos, internos e de download, este site é apenas um exercício que para fixação sobre os mesmos.



Links Internos

Os links internos são aqueles filiados ao seu site/ao documento atual, que não levam a sites de terceiros e podem ser uma parte anterior ou posterior do documento.

Um exemplo é adicionar uma segunda página.


Links Externos

Os links externos são aqueles que não pertencem a você, que não estão atrelados ao documento atual. Estes podem ser, por exemplo, um link para um site de um patrocinador.


Links de Download

Para faze-los, não requer muita coisa, apenas adicionar as tags download e type, colocando na ultima citada o tipo de media que se esta baixando (isso pode ser consultado no site isana.org).
