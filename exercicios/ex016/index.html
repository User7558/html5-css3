Como criar um degradê com CSS?

Alguns adendos...

A configuração global do site, ou seja, aquela com menor nível na hierarquia e que se aplica para todas as partes do site que não tenham uma configuração específica, é feita utilizando um *.

Exemplo: 

          *{
            aqui dentro ficam as configurações globais do site
          }

Código do gradiênte

          body{
            background-image: linear-gradient(90deg, yellow, red)
          }

Direções

  • 90deg ou to right →
  • to left
  • to top
  • to bottom
  • 45deg
  • -45deg
  • circle
