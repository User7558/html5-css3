Trabalhando com listas

No HTML existem vários tipos de listas (apenas 3, na verdade), e nesta aula veremos todas.


Listas Ordenadas

Onde a ordem é muito importante, podemos personalizá-las mudando sua formatação, ou mudando por onde ela começa.

  1. Acordar 5:30AM
  2. Ir para a escola
  3. Almoçar
  4. Estudar
  5. Jantar
  6. Ir para o curso
  7. Dormir

Listas não Ordenadas

Listas onde a sequência não é importante, igualmente customisáveis.

  • pão
  • leite
  • feijão
  • arroz
  • carne
  • queijo
  • frutas

Misturando Listas

É possível misturar esses dois tipos de listas, ou caso não queira, criar sub-listas do mesmo tipo, vejo o exemplo a seguir:

  1. item 1
  2. item 2
  3. item 3
    • item 3.1
    • item 3.2
  4. item 4

Listas de Definições

Como um dicionário, as mesmas tem termos e descrições/definições.

HTML
Linguagem de marcação utilizada para criar o conteúdo do site.
CSS
Lingaguem de marcação para a especificação de estilos em sites.
JavaScript
Linguagem de programação para criar interatividades em sites.
